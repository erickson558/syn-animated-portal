# Changelog

Todos los cambios relevantes de este proyecto se documentan aqui.

El formato esta basado en Keep a Changelog y versionado SemVer (`Vx.y.z`).

## [V1.1.2] - 2026-03-09

### Changed

- Se actualizo el acceso del portal principal de `Monitor Geko` a `MonitorApp`.
- La URL del acceso rapido ahora apunta a `http://localhost:888/monitoreos/monitorapp/`.
- Version de aplicacion incrementada a `V1.1.2`.

## [V1.1.1] - 2026-03-05

### Fixed

- Correccion de URL de Kameleon en el portal: de ruta raiz a `monitoreos/kameleon/index.html` para compatibilidad con EasyPHP.

## [V1.1.0] - 2026-03-05

### Added

- Intro de carga cinematografica con barra animada.
- Grid animado de ambiente para profundidad visual.
- Efecto de typing en el subtitulo principal.
- Cursor trail neon para escritorio.
- Sparkles animados en cada card.

### Changed

- Actualizacion de version de app a `V1.1.0`.
- Rehidratacion de sparkles al recargar tarjetas via AJAX.

## [V1.0.0] - 2026-03-05

### Added

- Badge de version visible en cabecera.
- Fondo ambiental con blobs animados.
- Footer con atajos de teclado y metadatos de version.
- Metadatos HTML de aplicacion (`application-name`, `description`).
- Documentacion base para GitHub (`README.md`, `LICENSE`, `VERSION`).

### Fixed

- Correccion del calculo del eje Y en el efecto tilt de tarjetas para mejorar la respuesta del puntero.
